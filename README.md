# Minesweeper Game

본 프로젝트는 **지뢰찾기(Minesweeper)** 개발을 통한 오픈소스소프트웨어를 실습하기 위해 생성되었습니다.
해당 프로젝트는 부경대학교 '오픈소스소프트웨어' 강의를 통해 진행되고 있습니다.

---

## 프로젝트 개요

* **프로젝트 주제**: 지뢰찾기 게임 개발을 통한 오픈소스소프트웨어 학습
* **목표**

  * 기존 지뢰찾기 게임의 핵심 규칙을 유지
  * 이슈를 통해 추가적인 기능을 구현
  * 타 학우와 PR 제출 및 코멘트 등의 과정을 통해 Git을 통한 오픈 소스 학습

---

## ✨ 주요 기능 및 추가 이슈

### 숫자 색상 표기 기능 (#2)
<img width="554" height="624" alt="text_color" src="https://github.com/user-attachments/assets/0fb778a7-f313-4a66-b13b-c5a0917adc06" />

게임 보드에 표시되는 숫자에 서로 다른 색상을 적용하여 시각적 구분을 강화했습니다.

| 숫자 | 색상 |
| -- | -- |
| 1  | 파랑 |
| 2  | 초록 |
| 3  | 빨강 |
| 4  | 남색 |
| 5  | 갈색 |
| 6  | 청록 |
| 7  | 검정 |
| 8  | 회색 |

* 숫자별 색상 차이를 통해 주변 지뢰 개수를 직관적으로 파악 가능

### 난이도 설정 기능 (#3)
<img width="330" height="400" alt="diff_easy" src="https://github.com/user-attachments/assets/a506327c-346f-4c04-87fb-1bb8d596a7ea" />
<img width="554" height="624" alt="diff_normal" src="https://github.com/user-attachments/assets/73aa1f95-4d2a-4acf-9019-f350c31d9baa" />

<img width="810" height="880" alt="diff_hard" src="https://github.com/user-attachments/assets/2fbaf201-1bb0-492b-9444-20a7ed83fc11" />

게임 시작 시 난이도를 선택할 수 있으며, 난이도에 따라 보드 크기와 지뢰 개수가 달라집니다.

| 난이도    | 보드 크기   | 지뢰 수 | 지뢰 비율 |
| ------ | ------- | ---- | ----- |
| Easy   | 9 × 9   | 10개  | 12.3% |
| Normal | 16 × 16 | 40개  | 15.6% |
| Hard   | 24 × 24 | 99개  | 20.6% |

* 난이도 변경 시 게임은 **자동으로 초기화**
* 난이도가 높아질수록 전략성과 난이도 상승

### 힌트 기능 (#4)
<img width="554" height="624" alt="hint" src="https://github.com/user-attachments/assets/c3dac7d7-7300-45de-8d1b-de7d58f8a1fe" />

플레이 도중 **지뢰가 없는 안전한 칸 1개를 하이라이트**하는 힌트 기능을 제공합니다.

* `H` 키를 누르면 힌트 사용 가능
* 힌트로 공개된 칸은 1초 동안 highlight

---

### 타이머 표기 기능 (#5)

게임 진행 시간을 실시간으로 표시하는 타이머를 추가했습니다.

* 첫 사용자 행동(클릭) 시 타이머 시작
* 게임 종료(승리 / 패배) 시 타이머 정지
* 가독성을 고려하여 **위치, 글자 크기 및 색상 개선**

---

### 점수 제공 기능 (#6)

게임의 도전 요소를 강화하기 위해 점수 시스템을 도입했습니다.

#### 기본 점수

* 새로운 안전 영역 **1칸 공개 시 5점**

#### 시간 보너스 점수

* 기본적으로 **플레이 시간 1초당 1점 감소**
* 난이도별 최대 시간 초과 시 **시간 보너스 미제공**

| 난이도    | 최대 시간 |
| ------ | ----- |
| Easy   | 5분    |
| Normal | 10분   |
| Hard   | 20분   |

* 빠르고 정확한 플레이일수록 높은 점수 획득 가능
